About Methodology

This system uses the Vector Space Model to represent each paper and query as vectors. Documents are ranked using cosine similarity over TF-IDF weighted vectors.

1. Vector Space Model

Each document is represented as a vector of term weights: D = (w1, w2, ..., wn). Query is also represented in the same space.

2. Term Frequency (TF)

Raw term frequency counts occurrences of a query term in a document: TF(t, d) = count of term t in document d.

3. TF-IDF

TF-IDF reduces the importance of very common terms: TF-IDF(t, d) = TF(t, d) * log(N / DF(t)), where N is total documents and DF(t) is document frequency.

4. Cosine Similarity

Similarity between query vector q and document vector d: cos(q, d) = (q . d) / (||q|| * ||d||). Higher values indicate stronger relevance.

5. Precision@K

Measures relevance quality in top-k results: Precision@K = Relevant Documents in top K / K. The dashboard shows precision progression from K=1 to K=10.

Example

For query "AI data learning", the engine preprocesses into tokens, computes TF in each document, transforms to TF-IDF vectors, ranks by cosine similarity, and evaluates precision across top results.
